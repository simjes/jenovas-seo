# Jenvoas Vitner SEO Project

## Requirements

Requires Spotify API credentials to run, see [env.example](.env.example) and [Spotify Developer Documentation](https://developer.spotify.com/documentation/web-api).

## Development

```base
npm install
npm run dev
```

## Background
